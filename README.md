# LPC824_300mil_board
LPC812上位バージョンLPC824をブレッドボードで使うためのボード  
以前作成した LPC812 300mil board を元に作成しました。  
  
・機能  
LPC824を300mil幅に変換  
リセットスイッチ  
ISPスイッチ  
電源LED搭載  
LチカLED搭載  
  
・パーツセット  
基板  
LPC824M201JDH20  
細いピンヘッダ20ピン x2  
タクトスイッチ x2  
チップLED x2  
チップ抵抗 2kΩ x2  
チップコンデンサ 0.1uF x2  
ハーフピッチ2列ピンヘッダ 5x2ピン
